Bizarre Sorting algorithms

This page contains some fun but useless sorting algorithms

Randomized Bogo Sort

This algorithm reorders the elements randomly and checks if the elements are in correct order until the array is sorted.

The fun part is, it has a very small chance to sort any array faster than (almost) any other algorithm.

Sleep sort

For each element in the array, the algorithm sleeps. The sleep duration is equal to the value of the element, so for value 1 it sleeps for 1 time unit (such as ms or s).

If the values are small, it can be really fast sorting method but if the array contains large values, this algorithm is unusable.
