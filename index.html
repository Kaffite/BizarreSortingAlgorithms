Bizarre Sorting algorithms

This page contains some fun but useless sorting algorithms

Randomized Bogo Sort

This algorithm reorders the elements randomly and checks if the elements are in correct order until the array is sorted.

The fun part is, it has a very small chance to sort any array faster than (almost) any other algorithm.

Sleep Sort

This algorithm uses a sleep function. The sleep duration is equal to the value of the element, so for a value of one, it sleeps for one time unit (such as one ms or s).

If the values are small, it can be really fast sorting method but it becomes impractical when the array contains large values.

Miracle Sort

This algorithm is known to be extremely reliably unreliable.

The Miracle sort does not actually do any sorting iself and hopes for a miracle to sort the array, hence the name.
